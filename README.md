# Spotify Data Analytics Capstone

## Project Overview

This project focuses on analyzing Spotify music data to uncover insights about music trends, user preferences, song characteristics, and factors influencing song popularity.

The analysis includes data cleaning, exploratory data analysis (EDA), visualizations, feature correlations, trend analysis, and predictive modeling techniques.

The goal is to better understand listener behavior and the characteristics that contribute to successful and popular songs.
